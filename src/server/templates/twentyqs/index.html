This is a chatbot that plays the "20 Questions" game. It was inspired by this tweet which notes how ChatGPT is unable to play 20 Questions due to lack of private 'memory', i.e. it can only 'remember' what has been written in the chat, it can't keep a secret.

It is powered by OpenAI (GPT-3.5) and written in Python (Langchain + Gradio UI).

The OpenAI API costs money, so for now there is no public demo.
If you want to try it out, DM me on Twitter and I'll send you a link.

Or host your own, or run it locally!
